### 5.2 错误分类

```
异常类型
     │
     ├── ToolUnavailableError (工具不可用)
     │     └─ 原因：缺少 API key, 环境未配置
     │
     ├── ToolExecutionError (工具执行失败)
     │     └─ 原因：参数错误，外部服务失败
     │
     ├── ApprovalRequiredError (需要审批)
     │     └─ 原因：危险命令，用户确认
     │
     ├── TimeoutError (超时)
     │     └─ 原因：网络延迟，长时间运行
     │
     └── RateLimitError (频率限制)
           └─ 原因：API 配额耗尽
```

## 6. 性能关键路径

### 6.1 延迟热点

| 阶段 | 典型延迟 | 优化策略 |
|------|----------|----------|
| LLM API 调用 | 500ms - 5s | 提示缓存，流式输出 |
| 工具注册 | 100-500ms | 懒加载，缓存 schema |
| 会话加载 | 10-100ms | SQLite 索引，限制历史长度 |
| 文件操作 | 1-50ms | 异步 I/O, 批量操作 |
| 网络请求 | 100ms - 2s | 连接池，重试退避 |
