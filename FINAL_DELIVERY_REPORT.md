## 五、部署指南

### 5.1 环境要求

- Python 3.10+
- pytest (测试框架)
- requests (HTTP 客户端)

### 5.2 快速安装

```bash
cd ~/hermes-agent
./scripts/install_hermes_enterprise.sh
```

### 5.3 验证安装

```bash
python -m pytest tests/enterprise/ -v
python scripts/monitor_hermes.py
```

### 5.4 配置监控

```bash
crontab -e
# 每小时执行健康检查
0 * * * * cd ~/hermes-agent && source .venv/bin/activate && python scripts/monitor_hermes.py
```
